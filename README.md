# 📚 Odin Project Library

A dynamic library management web app built with vanilla JavaScript as part of [The Odin Project](https://www.theodinproject.com/) curriculum. Add, remove, and track your reading progress with persistent in-memory storage.

## ✨ Live Features

- **Add Books** – Modal form with title, author, page count, and read status
- **Toggle Read Status** – Switch books between "Read" and "Not Read" with visual feedback
- **Remove Books** – Delete books from your collection
- **Visual Status Indicators** – Books change background color based on read status:
  - Blue-tinted background for read books
  - Red-tinted background for unread books
- **Unique IDs** – Each book gets a cryptographically secure UUID

## 🎨 Color Palette (60-30-10 Rule)

Following design principles:
- **60% Dominant** – Soft off-white background (#F7F0E5) for comfortable reading
- **30% Secondary** – Deep navy blue (#13274F) for headers and primary text
- **10% Accent** – Crimson red (#CE1141) for important action buttons

## 🛠️ Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- Vanilla JavaScript (ES6)
- Dialog API for modals
- UUID API for unique identifiers
